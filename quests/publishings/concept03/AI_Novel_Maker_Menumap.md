# AI Novel Maker Menu Map

## I. Main Navigation (Sidebar)

*   **AI Novel Maker (Logo/Home)**
    *   Link: `index.html` (or `dashboard.html` if it were the main dashboard)
    *   Description: Main entry point, displays user's projects.

*   **Dashboard (내 프로젝트)**
    *   Link: `index.html`
    *   Description: Overview of user's novel projects.
    *   Sub-items:
        *   Project Card (Clicking a project card)
            *   Link: (Implied, likely leads to `orchestra.html` or a project-specific workspace)
            *   Description: Opens the workspace for a specific project.

*   **Agent Management (에이전트 관리)**
    *   Link: `agents.html`
    *   Description: Manage AI agents for novel creation.
    *   Sub-items:
        *   Agent Edit Page
            *   Link: `agent-edit.html` (accessed by clicking an agent card)
            *   Description: Edit details of a specific AI agent.

*   **Templates (템플릿)**
    *   Link: `templates.html`
    *   Description: Browse and manage project and agent templates.

*   **Settings & Subscription (설정 및 구독)**
    *   Link: `settings.html`
    *   Description: User account, subscription, and application settings.
    *   Sub-sections (within `settings.html`):
        *   Subscription
        *   Billing
        *   Account
        *   API Keys
    *   Related Page:
        *   Pro Plan Details
            *   Link: `pro-plan.html` (accessed from settings page)
            *   Description: Information about upgrading to a Pro plan.
